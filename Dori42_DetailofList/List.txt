List에서의 Capacity의 의미는 배열의 크기가 아닌 메모리 할당 기준이다.
배열이 늘어나서 메모리 할당이 되었을 경우 자동으로 2배수로 늘어난다.

Enumerable은 List를 선언과 동시에 초기화를 수행한다.
(Enumerable.Repeat(5, 10)).ToList<int>()

List안의 요소들을 String 문자열으로 변경하는 메서드
string strListData = String.Join("구분자", 리스트);

List안의 모든 값을 변경
List<int> ChangedList = 리스트.ConvertAll<int>(iNum=> iNum * 3 );

for문을 통해 Add 할 경우 메모리 할당이 2, 4, 8, 16으로 커진다
List.Datasource를 통해 초기화 후 할당을 바로 할경우 메모리 할당이 숫자와 동일하게 할당된다
